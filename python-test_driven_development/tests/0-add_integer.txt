0-add_integer
=============

This file contains tests for the function add_integer.

First, import the function:

>>> add_integer = __import__('0-add_integer').add_integer

Simple addition:
>>> add_integer(1, 2)
3

Adding a positive and a negative number:
>>> add_integer(100, -2)
98

Using the default value for b:
>>> add_integer(2)
100

Using floats (they are cast to ints before addition):
>>> add_integer(100.3, -2)
98

Both arguments as floats:
>>> add_integer(3.9, 4.2)
7

TypeError when b is not a number:
>>> add_integer(4, "School")
Traceback (most recent call last):
TypeError: b must be an integer

TypeError when a is not a number:
>>> add_integer("School", 4)
Traceback (most recent call last):
TypeError: a must be an integer

TypeError when a is None:
>>> add_integer(None)
Traceback (most recent call last):
TypeError: a must be an integer
