Matrix division
====================

Basic division
--------------
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Original matrix should not change
---------------------------------
>>> matrix
[[1, 2, 3], [4, 5, 6]]

Floats and rounding
-------------------
>>> matrix_divided([[1.5, 2.5], [3.75, 4.25]], 2)
[[0.75, 1.25], [1.88, 2.12]]
