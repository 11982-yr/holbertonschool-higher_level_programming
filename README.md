# Higher Level Programming – Holberton School

This repository contains all Higher Level Programming projects completed as part of the Foundations v2.1 curriculum at Holberton School.

The projects span multiple programming languages and concepts including Python, SQL, data structures, object-oriented programming, and more.
Each directory represents a separate project with its own tasks and detailed README.

## Repository Structure
```bash
holbertonschool-higher_level_programming/
│
├── python-hello_world/
├── python-if_else_loops_functions/
├── python-import_modules/
└── (additional project folders as assigned)
```

## Projects Overview

Below is a summary of major project topics contained in this repository:

**Python – Hello, World**

Directory: ``` python-hello_world/ ```

This is the introductory Python project, covering:
- Running Python scripts
- Using print statements
- String manipulation
- Basic debugging
- Writing shell scripts that execute Python code

**Python – If/Else, Loops, Functions**

Directory: ```bash python-if_else_loops_functions/ ```

Covers:
- Control flow
- Loops
- Basic functions
- ASCII operations

**Python – Import & Modules**

Directory: ``` python-import_modules/ ```

Learn how Python’s import system works:
- Creating and importing modules
- Using functions across files
- Command-line arguments
- The __name__ == "__main__" pattern
- Inspecting modules with dir()


## General Requirements for All Projects

- All files compiled/interpreted on Ubuntu 22.04 LTS
- Python projects use Python 3.10+
- Code follows pycodestyle (PEP8)
- Files end with a newline
- First line of Python files:
```shell
#!/usr/bin/python3
```
- All scripts must be executable
- Each project contains its own README.md
