# Python ‚Äì Abstract Base Classes (ABC)

This project is part of the Higher Level Programming curriculum at Holberton School and focuses on abstraction, interfaces, and advanced object-oriented design in Python using Abstract Base Classes (ABCs).

The goal of this project is to understand how abstraction enforces structure in object-oriented programming and how Python supports interface-like behavior through the abc module, mixins, and duck typing.

## Learning Objectives
By completing this project, you should be able to:
* Understand abstraction and why it is useful
* Use the abc module to define abstract base classes
* Implement abstract methods
* Apply duck typing principles
* Create and use mixins
* Design flexible and extensible class hierarchies
* Track behavior across inherited classes

## Files Description

| File                         | Description                                                     |
| ---------------------------- | --------------------------------------------------------------- |
| `task_00_abc.py`             | Demonstrates the use of Abstract Base Classes                   |
| `task_01_duck_typing.py`     | Implements duck typing using shape-like classes                 |

| `task_05_dragon.py`          | Adds a Dragon class using flying and swimming mixins            |
| `README.md`                  | Project documentation                                           |
